# 孩子作业管理系统

一个帮助家长管理孩子作业的Web应用系统。

## 核心功能

- ✅ 任务管理（固定任务 + 动态任务）
- ✅ 时间规划与计时
- ✅ 积分奖励系统
- ✅ 森林成长可视化
- ✅ 统计分析

## 技术栈

- 前端：React 18 + TypeScript + Vite
- 后端：Node.js + Express
- 数据库：SQLite

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 项目结构

```
.
├── frontend/          # React 前端应用
├── backend/           # Express 后端服务
├── docs/              # 文档
└── package.json       # 根配置文件
```
